# 2026年2月的最后一天，春天终于来了

今天我恢复了我暂停了很久的blogs。在24年我喜欢写一些随笔和一些课程相关的笔记，随着事情日渐繁忙和PhD申请的磕磕绊绊，这个习惯和我的夜跑一样都慢慢消失了。

但是在2026年我决定重新拾起他们。经历了长达四个月的冰寒和一场席卷东海岸的冬季风暴，在2月的最后一天，DC终于迎来了春天的暖风。昨天又一路经过乔治城走过大桥，慢慢地觉得自己又想起了多年前在笼罩夜幕的珞珈山上吹着微微的凉风边跑步边思考人生何去何从。

从2024年以来，有很多坏消息，也有很多好消息。我拿到一些offer，从里面挑选了一个最合适的，然后在西湖待了三个月就匆匆飞往DC。当时觉得自己的申请策略比较失败，没有听家人的申一些MS作为过渡。所幸老板非常友善也很支持我的研究，想着就这样发些文章早些获取一个学位就好。

转眼已经是半年过去。本科的一些遗留的成果陆续有了回报，有时候不禁在想，如果这时我在读MS，沉淀一年的成果或许可以帮助我申请到一些更好的平台和学校。林中的路分为两条，选择了其中一条，就最好脚踏实地走下去。抱着这样的心态，我继续投身于忙碌的学习生活中。
