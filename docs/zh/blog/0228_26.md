# 2026年2月的最后一天，春天终于来了

今天我恢复了我暂停了很久的blogs。在24年我喜欢写一些随笔和一些课程相关的笔记，随着事情日渐繁忙和PhD申请的磕磕绊绊，这个习惯和我的夜跑一样都慢慢消失了。但在2026年的春天我决定重新拾起他们。

经历了长达四个月的冰寒和一场席卷东海岸的大雪之后，DC在2月的尾巴终于迎来了春天的暖风。昨天我又一路经过乔治城的街道，走过书店和大桥。风是一种回忆，桥上的凉风让我想起许多年前在长江大桥上散步和骑车的闲散时光，也想起在笼罩夜幕的珞珈山上边跑步边思考人生的方向。

从暂停博客以来，这一年多时间里有很多坏消息，也有很多好消息。我拿到一些offer，从里面挑选了一个最合适的，在西湖待了三个月后就匆匆飞往DC。当时觉得自己的申请策略比较失败，没有听家人的申一些MS作为过渡。所幸老板非常友善也很支持我的研究，想着就这样发些文章早些获取一个学位就好。我是一种随遇而安的性格，一直觉得人生的意义在于享受过程。

转眼已经是半年过去。本科的一些遗留的成果陆续收尾，而持续的努力也产出了一些结果。有时候会想这时我在读MS，沉淀一年的成果或许可以帮助我申请到一些更好的平台和学校。林中的路分为两条，选择了其中一条，就最好脚踏实地走下去。风是一种回忆，但也只是一些回忆，人生的过程就在于选择了林中的路之后，好好享受眼前的风景。
