# 如何度过夜晚时光？

念书总共要克服两个难关。第一是研究上的难题。在Agent实用性大大提升的当下，Writing和Coding已经不再像以前那样耗费时间，那么研究的taste就成为了一种可贵的品质。有时候因为切入问题的角度不对，或者是钻进某个死胡同，接连数周得不到进展会让心情非常焦虑，漫漫长夜难以入睡。

那么该如何度过夜晚？笔者觉得需要培养一个好的习惯，不断学习总结其他论文的角度尤其是多看一些社区评价高的论文是非常有好处的。鉴于笔者是一位非常junior的博士生，没有办法给出实际的操作步骤，只能分享一些个人经验。笔者会从近期的会议里通过关键词检索搜集多份论文清单，针对每份清单会花一至两天从20-30篇中挑选出5篇认为比较好/优雅的论文，然后细细研读。在2025年笔者第一次这样做的时候，每份筛选会花费3-4天，而现在大概是1-2天，加速比~50%。

第二个难关是对抗内心的孤独。笔者24年在berkeley访问时，因为学期结束和暑研之间相隔了一个多月，于是没课之后每天早起去很远的图书馆自习到深夜，保持了长达一个月的零社交。这只是一个微不足道的例子，实际上博士生的孤独很大程度上来自于长时间的低社交能量和独自一人handle所有事情的无奈感。偶尔能刷到一些博士生求对象的帖子，一方面是空虚孤独的内心急需另一个灵魂来依靠和倾诉，一方面是至少生活上能够得到脱离孤独的能量。

笔者是一位非常乐观的ISTJ，能够非常好地管理消极情绪，但有时也会觉得难以度过某个夜晚时光。为了缓解孤独情绪，暂时推荐两个方法。第一是出去散心，并且听一些轻松的jazz。笔者关注了很多jazz的轻音乐歌单，原本是用于白天工作时当作背景音乐的，但是意外的发现这些音乐对于缓解孤独焦虑很有效。第二是看一些电影/电视剧/动漫。昨天是周六，发现周边人都有社交活动，于是感到心情非常糟糕，为了缓解这种孤独焦虑，笔者一晚上时间重温了进击的巨人全系列，睡前感觉心情好多了。今天发现情绪仍有一些滞留，于是在学校看完了整部相合之物，内心感动之余觉得应该写下来这种缓释心情的好方法。

夜晚是美好的，夜晚是孤独的。夜晚是所有人的，夜晚也只是你自己的。可能大多数成年人都要慢慢学会如何度过夜晚的漫漫时光。
